1. Dictionary

Class: rbn-node

Class precedence list: rbn-node, standard-object, t

The Random-Boolean Network class.

Function: initialize n class @&rest args

returns n rbn-node instances in an array.

Function: investigate-rbn nodes

creates a graph

Function: make-random-global-states 2^n

returns an (integer 0 n) which is randomized for the initial state.

Function: make-random-mask n k

returns an (integer 0 n) which has only k bits on.

Function: make-rbn-graph 2^n

create the container of the graph which represents the propagation of states. n is a number of nodes. returns array of size = 2^n .

Function: output-graph-to-csv graph @&optional path

output the result graph of an rbn states to the specified path as a csv file.

Function: output-nodes-to-csv nodes @&optional path

output the node connection of an rbn to the specified path as a csv file.

Function: print-integer-binary a

print an integer in binary format

Function: propagate node states

propagates each node. Uses global set of states as its input.

Function: propagate-all nodes states

propagates all nodes and return a global state in the form of integer (bit-vector).
